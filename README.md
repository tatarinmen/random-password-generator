# Random Password Generator (Генератор случайных паролей)

##  Автор
Тазетдинова Карина

## 📝 Описание
Графическое приложение на Python для генерации надёжных паролей с настраиваемыми параметрами. Поддерживает выбор длины, типов символов, автоматическое сохранение истории генераций в формат JSON и быстрое копирование в буфер обмена.

### ✨ Возможности:
-  Ползунок для выбора длины пароля (4–32 символа)
-  Чекбоксы: Буквы, Цифры, Спецсимволы
-  Таблица истории с временными метками
-  Автосохранение/загрузка истории в `password_history.json`
-  Кнопка быстрого копирования
-  Валидация ввода и защита от пустых конфигураций
